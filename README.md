# CODM-PROJECT
Plateforme de tournois Call of Duty Mobile (CODM) Application web développée avec Django permettant d’organiser et de gérer des tournois Call of Duty Mobile. Les joueurs peuvent s’inscrire, participer aux tournois et soumettre les résultats des matchs, tandis que les administrateurs gèrent les participants, les matchs et le déroulement des tournois
